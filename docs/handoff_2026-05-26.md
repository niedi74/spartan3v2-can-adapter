# Spartan3v2 CAN Adapter - Handoff (2026-05-26)

## Zielbild aktuell
- Spartan ESP arbeitet als BLE Gateway:
  - BLE Peripheral fuer M5/Waveshare (`Spartan3-Hub`)
  - BLE Central zur 123TUNE+ (NUS)
- Datenpfad: 123TUNE + Spartan Lambda -> Spartan BLE Status Payload -> M5 Gateway Mode

## Wichtige Ergebnisse aus dem Live-Chat
- Stabiler Fortschritt erreicht:
  - 123TUNE wird nach Zuendung-Reset wieder gefunden.
  - Subscribe/CCCD/Wake (`$` + `CR`) funktionieren.
  - M5-Hub-Advertising wird erst nach 123-Connect (oder nach Fallback-Zeit) gestartet, damit 123-Connect priorisiert wird.
- Beobachtet:
  - Teilweise nur `0D` Notifies ohne volle Live-Frames, wenn Motor nicht laeuft bzw. 123 nicht sauber streamt.
  - Mit Motorlauf ist der entscheidende Realtest fuer stabile RPM/ADV/MAP-Frames.
