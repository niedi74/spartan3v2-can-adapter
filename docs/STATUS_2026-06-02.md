# Spartan + M5 Dial — Status 2026-06-02

## Architektur

```
123TUNE+ (BLE)     ─┐
Spartan 3 v2 (CAN) ─┤  ESP32 "Spartan3-Hub"  ──BLE──>  M5 Dial (Gateway)
BM6 Battery (BLE)  ─┤  Web GUI (Live|Setup)             8 Display-Pages
Reed Speed (GPIO27)─┘  COM9 / 115200
```

## BLE Gateway Payload (Notify 250ms)

```
L<lambda>R<rpm>A<adv>M<map>V<bm6_volt>S<speed_kmh>I<123_volt>T<123_temp>C<coil_A>
```

Alle Felder nach M sind optional (nur gesendet wenn Quelle verbunden).

## Spartan Web-GUI

Zwei Tabs: **Live** und **Setup**.

### Live
- Lambda Hero-Card (Wert, Source, Status, Temp, 123 RPM/ADV/MAP)
- 60s Charts (Lambda, RPM, Temp)
- 123Tune BLE Diagnose
- BM6 Batteriemonitor (Spannung, Temp, RX)
- Geschwindigkeit Reed (Hz, km/h, Trim-Form)

### Setup
- System Diagnose (CAN, Heap, AP IP, JSON Dump)
- BLE Hub / Gateway (UUIDs, Clients)
- WLAN / Hotspot (Presets, manuell)
- Spartan UART (Befehle, CAN-Config, Betriebsart)
