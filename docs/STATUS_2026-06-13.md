## Hub — was `main` kann

- **Lambda** per CAN (Spartan 3 v2), UART-Konfig, optional Demo/Analog
- **123TUNE+** per BLE Central (NUS), Reconnect-State-Machine, Radio-Priorität
- **BM6** Batterie-Monitor (dual-slot main/aux)
- **Cockpit → Displays** per **ESP-NOW Broadcast** (250 ms, 14-Byte-Frame + CRC-8)
- **BLE Display Server aus** (`ENABLE_BLE_DISPLAY=0`) — kein instabiles Multi-GATT mehr
- **Event-Log:** `GET /log/events`, `/log/events.csv`
- **CSV Fahrt-Log** auf SPIFFS
- **NTP / time_epoch** — Hub als Zeitmaster
- **Reed-Speed** GPIO 27, Heater-ADC GPIO 35
- **Live-Debug:** Handy-Hotspot `Android-AP1` → `docs/road-live-debug.md`

## Build-Flags (`motorraum`)

```
ENABLE_BLE_HUB=1          # 123TUNE BLE Central
ENABLE_BLE_DISPLAY=0      # Displays NICHT per BLE
```
